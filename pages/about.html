Image Search Abstraction Layer

(a Free Code Camp API project)

This microservice (which is built on top of Google custom search) provides a simple API for image search.

Usage

  • API endpoint: https://imsearch.herokuapp.com/search
  • Response format: JSON
  • Parameters:
    • q [required]: search query
    • offset [optional]: how many results will be skipped, counting from the first one
  • GET-request example: https://imsearch.herokuapp.com/search?q=foo&offset=5
  • Response example:
    [ { url: "https://upload.wikimedia.org/wikipedia/commons/c/cb/Foo_was_here.jpg", snippet: "Foo was here", thumbnail: "https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcThc2UCAedSGYuBtMtcCdep9wp7wndetyRUsSXvFnRoat_Buw5s1OK-jQ", context: "https://en.wikipedia.org/wiki/Foo_was_here" }, //... another 9 objects ]
  • To get last 10 submitted search queries issue a GET-request to https://imsearch.herokuapp.com/latest
